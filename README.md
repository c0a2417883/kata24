# 片柳学園フレッシャーズロボコン2024 制御システム

## 概要
片柳学園フレッシャーズロボコン2024の競技テーマ「四目並べ」に出場した手動操縦ロボットの制御ファームウェアです。
本競技では、タワーに対して自チームのコマ（ボール）を素早く投入し、盤面を揃えるスピードと戦略が求められます。

本システムでは、専用のラジコン用送信機を使用せず、ESP32をWi-Fiアクセスポイント兼Webサーバーとして稼働させ、スマートフォンやPCのブラウザからWebSocket経由でゲームパッドの操作入力を低遅延で送信するIoTベースの操作システムを構築しました。また、操縦者がコマの投入（機構操作）に集中できるよう、IMUを用いた機体の自動姿勢維持（ヨー角のPID制御）アシスト機能を実装しています。

## システム構成・技術スタック
ESP32をメインマイコンとし、通信からハードウェア制御までを単一のチップで非同期処理しています。
* **Microcontroller:** ESP32 (Arduino環境)
* **Communication:** Wi-Fi (SoftAPモード), WebSocket (`ESPAsyncWebServer`), HTML5 Gamepad API
* **Sensors & Actuators:**
  * IMUモジュール: BNO055 (I2C通信)
  * 足回り: 3輪オムニホイール (自作のキネマティクスライブラリ)
  * 機構部: ボール巻き込み・射出用DCモーター
* **Languages:** C++, HTML, JavaScript

## 実装した主要機能
- **WebSocketによる超低遅延リモートコントロール**  
  ESP32内にWebサーバーを立ち上げ、操作用のHTML/JSをホスティングしています。クライアント（PC等）で接続したゲームパッドの入力値を10ms周期で取得し、WebSocketを通じてESP32へバイナリデータとして送信することで、タイムラグを感じさせない滑らかな操作を実現しました。

- **IMUとPID制御による姿勢維持アシスト**  
  オムニホイール機体特有の「直進時の意図しない回転（ふらつき）」を防ぐため、BNO055から取得したヨー角（Yaw）の偏差をPID制御器に入力し、目標角度を自動で維持するHeading Lock機能を実装しています。旋回時のみ目標角度を更新することで、直感的な操縦と安定性を両立しました。

- **3輪オムニホイールの運動学演算**  
  ジョイスティックからのX/Y方向の並進速度指令と、PID制御器からの角速度指令を合成し、3つのモーターそれぞれの目標回転速度へ変換する処理（`diffdrive.h`）をゼロから実装しています。
